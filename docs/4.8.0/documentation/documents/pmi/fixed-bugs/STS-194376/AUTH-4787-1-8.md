# ВМ. Актуальный UI при накатке на старую версию IAMProxy. Корректно перезатираются файлы в /opt/iamproxy/rds-client/templates

## Описание

При накатке на ВМ на старую версию в /opt/iamproxy/rds-client/templates оставались старые файлы, которые ломали UI, в частности некорректно перезатирался html/index.html , и из-за оставался старый UI.

## Предусловия

Наличие ВМ с версией IAMProxy ниже 4.0.0

## Шаги проверки

1. **Действие**:

   Произвести установку IAMProxy версии 4.8.0+ на ВМ из предусловия

   **Успешный результат**:

   Установка успешно выполнена

2. **Действие**:

   На ВМ перейти в папку /opt/iamproxy/rds-client/templates

   **Успешный результат**:

   В папке нет неактуальных файлов
    html/index.html обновлен

3. **Действие**:

   Перейти на страницу прокси и пройти аутентификацию

   **Успешный результат**:

   Аутентификация успешно пройдена, на стенде новый дизайн
