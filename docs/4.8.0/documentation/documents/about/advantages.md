# Преимущества

**Platform V IAM SE (IAM)** обладает следующими ключевыми особенностями и преимуществами по сравнению с аналогичными решениями:

- **IAM Proxy (AUTH)**:
  - Реализованы индустриальные протоколы аутентификации и авторизации OpenID Connect, OPA.
  - Высокая надежность и производительность.
  - Работа на вендорозамещенной инфраструктуре (ЗОКИИ Ready).
  - Поддержка требований надежности (активный healthcheck, метрики Prometheus, SberCA, Rate Limiters).
  - Поддержка требований безопасности и DevSecOps.
  - Поддержка альтернативных IDP (OIDC провайдеров): возможность настройки дополнительного провайдера аутентификации (например, ЕСИА, VK, Google) через параметр `oidc_alt_idp_discovery_url`, что позволяет реализовать гибридную или резервную аутентификацию.
  - Поддержка современных протоколов: полная поддержка HTTP/2 и gRPC, включая проксирование стриминговых запросов, health-check и балансировку нагрузки для gRPC-сервисов (указывается через `protocol: grpc` в конфигурации junction).
- **KeyCloak SE**:
  - Многофакторная аутентификация.
  - Интеграция с ЕСИА, VK, Одноклассники, Mail.ru.
  - Возможность настройки гранулированной ABAC/RBAC модели авторизации.
