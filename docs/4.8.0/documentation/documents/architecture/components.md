# Компоненты

| Название компонента                 | Ответственность компонента                                                                                                                                                                                                                                                                                                                                                      |
|-------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| RDS Client                          | Формирование конфигурационных файлов для IAM Proxy в части маршрутизации                                                                                                                                                                                                                                                                                                        |
| Proxy Server (IAM Proxy)            | Обеспечение аутентификации пользователей (в частности выполнение роли OIDC клиента в сценарии аутентификации OpenID Connect Code Flow). Выполнение предварительной авторизации обращения пользователей к прикладным сервисам. Выполнение функций проксирования и балансировки HTTP запросов (reverse proxy). Обеспечение архитектурного pattern Policy Enforcement Point (PEP). |
| Fluent Bit (Сервис обработки логов) | Передача журналов в подсистему централизованного аудита Platform V Audit SE                                                                                                                                                                                                                                                                                                     |
