# Выход из системы

**Основной сценарий**

1. Аутентифицированный пользователь завершает работу в UI приложения системы, доступ к ресурсам которой предоставляются
   через IAM Proxy.
2. Приложение системы производит завершение сессии по пользователю, и перенаправляет на относительный URI завершения
   сессии на IAM Proxy.
3. IAM Proxy очищает данные по пользователю, отзывает токены в провайдере аутентификации и перенаправляет пользователя в
   провайдер аутентификации для выполнения выхода из сессии.
4. Провайдер аутентификации закрывает сессию по пользователю и перенаправляет пользователя обратно на IAM Proxy.
5. Сеанс работы в системе завершен.
