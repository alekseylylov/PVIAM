# Упрощение пользовательского опыта (UX)

## Контекст

Сложные системы управления доступом могут вызывать трудности у конечных пользователей и администраторов.

## Проблема

Паттерн направлен на решение следующих проблем:
- долгий процесс настройки ролей и прав доступа;
- сложность понимания интерфейса для новичков.

## Решение

Platform V IAM SE предоставляет:
- интуитивный интерфейс с готовыми шаблонами ролей и политик;
- автоматизацию повторяющихся задач.
