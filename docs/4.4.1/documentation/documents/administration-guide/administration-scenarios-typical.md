# Типовые сценарии администрирования компонента IAM Proxy

Типовые сценарии администрирования:

- добавление ответвлений;
- удаление ответвлений;
- изменение ответвлений;
- контроль событий системного журнала. Описан в разделе [События системного журнала](system-log.md);
- контроль событий мониторинга. Описан в разделе [События мониторинга](monitoring.md).
