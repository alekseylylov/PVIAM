# Формат времени в стандартном access логе. [YYYY-MM-DD H:M:S.MS+03:00]

## Описание

Переход с формата [12/Dec/2025:15:39:39 +0300] на формат [2023-12-12 15:30:45.123+03:00]

## Шаги проверки

1. **Действие**:

   В сервисе DAPP перейти на вкладку Workloads -> Pods

   **Успешный результат**:

   Открыт список подов

2. **Действие**:

   Выбрать под iamproxy и перейти на вкладку Logs

   **Успешный результат**:

   В списке подов есть iamproxy, успешный переход на страницу логов пода

3. **Действие**:

   Посмотреть логи iam proxy

   **Успешный результат**:

   Формат даты в логах выглядит следующим образом: [YYYY-MM-DD H:M:S.MS+03:00]
