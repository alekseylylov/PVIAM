# Расчет сайзинга компонентов

## Методики расчета сайзинга для компонентов Platform V IAM

Для расчета необходимых КТС каждого компонента Platform V IAM необходимо руководствоваться информацией,
предоставленной в документации соответствующего компонента:

| Код компонента | Имя компонента                        | Расположение                                           |
|----------------|---------------------------------------|--------------------------------------------------------|
| AUTH           | IAM Proxy                             | `/documentation/reports/AUTH-load-testing-report.json` |
| KCSE           | KeyCloak.SE                           | `/documentation/reports/KCSE-load-testing-report.json` |
