# Модули программного компонента

| Название модуля                          | Ответственность модуля                                                                      |
|------------------------------------------|---------------------------------------------------------------------------------------------|
| ufs-platform-auth-api                    | Предоставляет базовое API модуля                                                            |
| ufs-platform-auth-impl                   | Содержит реализации целевых провайдеров аутентификации                                      |
| ufs-platform-auth-legacy                 | Содержит реализации специфичных провайдеров аутентификации                                  |
| ufs-platform-auth-channel                | Содержит реализации Uko и GuestUko провайдеров аутентификации для версий с постфиксом "-hl" |
| ufs-platform-auth-spring                 | Содержит реализации целевых провайдеров аутентификации на SpringFramework                   |
| authentication-spring-security-starter   | Стартер для приложений, разрабатываемых на SpringFramework                                  |
| authentication-spring-boot-starter       | Стартер для приложений, разрабатываемых на Spring Boot                                      |
| authentication-spring-boot-autoconfigure | Содержит автоматическую конфигурацию для клиентского модуля                                 |
