# Поддержка удаленной работы

## Контекст

Рост популярности удаленной работы требует надежных решений для безопасного доступа к корпоративным ресурсам извне.

## Проблема

Паттерн направлен на решение следующих проблем:
- сложности с безопасным подключением к внутренним системам из интернета;
- риск утечки данных через нешифрованные соединения.

## Решение

Platform V IAM SE предоставляет:
- single sign-on (SSO) для удаленных пользователей;
- интеграцию с SAML/OAuth2 для внешних провайдеров;
- защиту через MFA и шифрование трафика.
