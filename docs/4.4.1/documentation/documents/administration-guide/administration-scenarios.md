# Сценарии администрирования

## Введение

В разделе описаны варианты и типовые сценарии администрирования IAM Proxy.

> В разделе приведены названия переменных одинаково применимых для различных сред контейнеризации, указанных в
> системных требованиях. Имя переменной не определяет конкретную среду контейнеризации.
