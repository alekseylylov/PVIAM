# Настройки параметров безопасности

Сервис устанавливается, конфигурируется в соответствии с эксплуатационной документацией (Руководство по установке,
Руководство по системному администрированию). Параметры настройки размещены в разделе Руководство по установке.

Не допускается внесение изменений в конфигурационные файлы уже развернутых компонентов.
