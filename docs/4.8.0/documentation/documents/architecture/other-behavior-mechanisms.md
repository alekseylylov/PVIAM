# Прочие поведенческие механизмы

IAM Proxy предоставляет возможность использовать в своих веб-приложениях следующие возможности:

- `WebSocket` и `Server-send events`;
- взаимодействие с внешним провайдером аутентификации по `OpenID Connect`;
- авторизация доступа к защищаемым приложениям по ролям: дает возможность ограничить обращение к бэкенд приложениям на
  основе ролей из токена (по маске роли, регулярное выражение);
- балансировка (hash по `username`+`real ip`);
- хранение HTML-статики.

На IAM Proxy кешируются метаданные провайдера аутентификации `OpenID Connect` (время жизни кеша 24 часа).
Метаданные включают (данные получаемые от провайдера) описание возможностей и точек входа в провайдер аутентификации.

Пользовательские данные на IAM Proxy не кешируются.
