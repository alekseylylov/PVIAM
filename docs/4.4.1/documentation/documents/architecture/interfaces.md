# Программные интерфейсы

## Общая информация
IAM Proxy предоставляет набор программных интерфейсов (API) для взаимодействия с внешними системами, управления конфигурациями и мониторинга состояния. Все интерфейсы разделены на **публичные** (для интеграции с приложениями) и **внутренние** (для диагностики и администрирования). 

## Идентификаторы и названия интерфейсов

| Идентификатор интерфейса   | Название интерфейса | Компоненты, реализующие интерфейс |
|----------------------------|---------------------|-----------------------------------|
| `Interface_OIDC_Session`   | Управление сессиями | Proxy Server                      |
| `Interface_OIDC_Token`     | Работа с токенами   | Proxy Server                      |
| `Interface_RDS_Config`     | Конфигурация RDS    | RDS Server                        |
| `Interface_Logs`           | Логирование         | Сервис обработки логов            |
| `Interface_Testing`        | Тестирование        | Proxy Server                      |


### Interface_OIDC_Session  
**Назначение**: Управление сессиями пользователей (логин, логаут, реаутентификация).

| Название операции         | Входные параметры | Выходное значение                |
|---------------------------|-------------------|----------------------------------|
| Завершение сессии         | -                 | Статус: `{"status": "success"}`  |
| Ре-аутентификация         | -                 | Перенаправление на IDP           |
| Смена организации         | `org_id`          | Статус: `{"status": "changed"}`  |

### Interface_OIDC_Token  
**Назначение**: Работа с JWT-токенами (обновление, отзыв, получение offline-токена).

| Название операции         | Входные параметры      | Выходное значение                          |
|---------------------------|------------------------|--------------------------------------------|
| Обновление токена         | -                      | Новый JWT-токен                            |
| Получение offline-токена  | `scope=offline_access` | Offline-токен                              |
| Отзыв offline-токена      | `token_id`             | Статус: `{"status": "revoked"}`            |


### Interface_RDS_Config  
**Назначение**: Получение активной конфигурации RDS Server.

| Название операции         | Входные параметры | Выходное значение                |
|---------------------------|-------------------|----------------------------------|
| Получение конфигурации    | -                 | JSON-файл с текущими настройками |

### Interface_Logs  
**Назначение**: Прием и передача логов для анализа.

| Название операции         | Входные параметры       | Выходное значение             |
|---------------------------|-------------------------|-------------------------------|
| Прием логов               | Текстовые сообщения     | Подтверждение получения       |
| Передача метрик           | Метрики в формате HTTP  | Статус: `{"status": "sent"}`  |


### Interface_Testing  
**Назначение**: Диагностика и тестирование функционала IAM Proxy. Доступны только при включении опций развертывания.

| Название операции         | Входные параметры | Выходное значение                                 |
|---------------------------|-------------------|---------------------------------------------------|
| Проверка состояния        | -                 | Статус: `{"status": "healthy", "version": "..."}` |
| Отображение запросов      | -                 | Заголовки и тело HTTP-запроса                     |
| Получение токена          | -                 | Текущий OIDC-токен                                |
| Вывод метрик              | -                 | Метрики в формате Prometheus                      |
