# Соответствие нормативным требованиям

## Контекст

Организации обязаны соблюдать нормативные акты (№ 152-ФЗ, ISO 27001, PCI DSS), которые предъявляют строгие требования
к защите персональных данных и аудиту.

## Проблема

Паттерн направлен на решение следующих проблем:
- сложность сбора и анализа логов для аудита;
- риск нарушения законодательства из-за неправильной политики доступа.

## Решение

Platform V IAM SE предоставляет:
- централизованное логирование всех действий с учетными записями;
- генерацию отчетов для аудиторских проверок;
- настройку политик доступа в соответствии с нормативными требованиями.
