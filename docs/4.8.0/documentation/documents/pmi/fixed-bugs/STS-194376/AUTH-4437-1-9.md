# Браузер не использует для дублирования закэшированную страницу с истекшим сроком действия

## Описание

Включено задание Cache-Control если этого заголовка нет в ответе, и если это не редирект и если content-type ответа в списке потенциально содержащих чувствительные данные

## Шаги проверки

1. **Действие**:

   Открыть новую вкладку в режиме инкогнито в браузере на движке хромиума

   **Успешный результат**:

   Новая вкладка открыта

2. **Действие**:

   Перейти на страницу прокси

   **Успешный результат**:

   При переходе на страницу прокси происходит редирект на страницу аутентификации

3. **Действие**:

   Ввести валидный логин/пароль и нажать кнопку "вход"

   **Успешный результат**:

   Аутентификация успешно пройдена

4. **Действие**:

   После успешной аутентификации нажать ПКМ на вкладку с прокси и нажать кнопку "Дублировать"

   **Успешный результат**:

   При переходе на дублированную страницу открывается страница прокси (та, которую мы дублировали)
