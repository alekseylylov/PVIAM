# ВМ. Актуальный UI при накатке на новой версии IAMProxy корректно перезатираются изменененные файлы в /opt/iamproxy/rds-client/templates

## Описание

При накатке на ВМ на старую версию в /opt/iamproxy/rds-client/templates оставались старые файлы, которые ломали UI, в частности некорректно перезатирался html/index.html , и из-за оставался старый UI.

## Предусловия

Наличие доступа на ВМ с IAMProxy

## Шаги проверки

1. **Действие**:

   На ВМ, где установлен IAMProxy перейти в папку /opt/iamproxy/rds-client/templates

   **Успешный результат**:

   Установка успешно выполнена

2. **Действие**:

   На ВМ перейти в папку /opt/iamproxy/rds-client/templates и изменить содержимое какого-либо файла

   **Успешный результат**:

   Содержимое успешно изменено и сохранено

3. **Действие**:

   Произвести установку IAMProxy версии 4.8.0+ на ВМ из предусловия

   **Успешный результат**:

   Успешная установка

4. **Действие**:

   Перейти в папку /opt/iamproxy/rds-client/templates и проверить измененный файл

   **Успешный результат**:

   Файл был перезатерт после установки новой версии
