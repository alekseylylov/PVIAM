# Преимущества

**IAM Proxy (AUTH)** обладает следующими ключевыми особенностями и преимуществами по сравнению с аналогичными решениями:

1. Реализованы индустриальные протоколы аутентификации и авторизации OpenID Connect.
2. Высокая надежность и производительность.
3. Работа на вендорозамещенной инфраструктуре.
4. Поддержка требований надежности (активный healthcheck, метрики Prometheus, PKI CA, Rate Limiters).
5. Поддержка требований безопасности и DevSecOps.
