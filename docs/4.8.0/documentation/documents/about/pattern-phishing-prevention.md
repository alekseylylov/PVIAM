# Предотвращение фишинга и кражи учетных данных

## Контекст

Фишинг и атаки на кражу учетных данных остаются одной из основных угроз безопасности корпоративных систем.

## Проблема

Паттерн направлен на решение следующих проблем:
- пароли легко взламываются через фишинговые атаки;
- отсутствие дополнительного фактора аутентификации повышает риск компрометации учетных записей.

## Решение

Platform V IAM SE предоставляет:
- многофакторную аутентификацию (MFA): пароль + OTP/аппаратный ключ;
- интеграцию с HSM для защиты криптографических операций.
